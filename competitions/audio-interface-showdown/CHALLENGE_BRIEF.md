# BossCrafts Audio Interface Showdown\n\n## Goal\nDesign and build a modern, complete, browser-based audio interface with meaningful features and polished UX.\n\n## Submission Rules\n- Build a standalone frontend app using only HTML, CSS, and JavaScript.\n- No build tooling required to run.\n- Entry file must be `index.html`.\n- Include all assets in the submission folder.\n- Must run by opening `index.html` directly in browser.\n\n## Required Features\n- Play/pause and stop controls\n- Volume control and mute toggle\n- Seek/progress bar with current time and duration\n- Playlist support for multiple tracks\n- Keyboard shortcuts (`Space`, `ArrowLeft`, `ArrowRight`, `M`)\n- Visual audio feedback (waveform bars, spectrum bars, or equivalent)\n- Responsive layout for desktop and mobile\n- Distinct visual theme and intentional typography\n\n## Nice-to-Have Features\n- Loop and shuffle\n- Playback speed\n- Drag-and-drop audio file loading\n- Theme switching\n- Recently played list\n\n## Deliverables\n- `index.html`\n- `styles.css`\n- `app.js`\n- `README.md` with feature list, keyboard shortcuts, and run instructions\n\n## Judging Criteria\n- Completeness (40%)\n- Feature quality and reliability (30%)\n- UX and visual design quality (20%)\n- Code clarity and maintainability (10%)\n
