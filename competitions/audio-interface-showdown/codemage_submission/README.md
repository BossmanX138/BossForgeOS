# Aether Deck - Audio Interface Showdown Submission

A complete standalone browser audio interface built with plain HTML, CSS, and JavaScript. It runs by opening index.html directly in a browser and needs no build tooling.

## Implemented Features

- Play, pause, stop, previous, and next controls
- Volume slider and mute toggle
- Seek/progress bar with current time and total duration
- Playlist with multiple tracks
- Keyboard shortcuts: Space, ArrowLeft, ArrowRight, M
- Visual audio feedback via live spectrum bars
- Responsive layout for desktop and mobile
- Distinct visual theme with intentional typography and motion

## Extra Features Included

- Shuffle mode
- Loop mode
- Playback speed selection
- Drag-and-drop audio file loading
- File picker for local audio loading
- Recently played list
- Built-in generated demo tracks so the app works immediately without external assets

## Keyboard Shortcuts

- Space: Play or pause
- ArrowLeft: Seek back 5 seconds
- ArrowRight: Seek forward 5 seconds
- M: Mute or unmute

## Run Instructions

1. Open index.html in any modern desktop or mobile browser.
2. Press Play to start built-in demo tracks.
3. Drag and drop your own audio files into the drop zone, or click Choose Files.

## Notes

- No external libraries are used.
- No build step is required.
- All logic is self-contained in app.js.
